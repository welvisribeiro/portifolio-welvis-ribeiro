# 🎨 Recriação do Neumorphism.io

## 📝 Descrição do Projeto

Este projeto consiste na recriação da interface do site Neumorphism.io utilizando ferramentas de Inteligência Artificial (IA Studio), com o objetivo de aplicar conceitos modernos de design e desenvolvimento web.

O foco principal foi reproduzir o estilo visual conhecido como **neumorfismo**, que utiliza sombras suaves, profundidade e elementos minimalistas para criar uma interface limpa e moderna.

O projeto foi desenvolvido como atividade acadêmica, visando aprimorar habilidades em front-end, design de interfaces e uso de ferramentas de IA no desenvolvimento.

---

## 🚀 Tecnologias Utilizadas

* HTML
* CSS
* JavaScript
* IA Studio

---

## 📊 Resultados e Aprendizados

Durante o desenvolvimento deste projeto, foi possível:

* Compreender melhor o conceito de design neumórfico
* Praticar a construção de interfaces modernas
* Aprender a utilizar ferramentas de IA no desenvolvimento web
* Melhorar a organização e estruturação de código
