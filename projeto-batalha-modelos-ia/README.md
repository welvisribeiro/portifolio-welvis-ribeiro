# 🤖 Batalha de Modelos de IA & Engenharia de Prompt (XML)

## 📝 Descrição do Projeto

Este projeto consiste em uma análise comparativa entre diferentes modelos de linguagem (LLMs), avaliando sua capacidade de interpretar e executar um prompt estruturado em XML para geração de uma página HTML com CSS integrado.

Foram testados diversos modelos, incluindo ChatGPT, Gemini, DeepSeek, Qwen, Grok, Maritaca e Claude, com o objetivo de identificar diferenças de comportamento, precisão e consumo de tokens.

---

## 🎯 Objetivo

- Avaliar a fidelidade das IAs ao seguir um prompt estruturado  
- Comparar a precisão técnica na geração de código HTML  
- Analisar diferenças de desempenho entre modelos  
- Desenvolver pensamento crítico sobre uso de IA no desenvolvimento  

---

## 🧪 Metodologia

### 📌 Prompt Utilizado (XML)

```xml
<tarefa>
    <objetivo>Criar uma página HTML5 única com CSS3 interno (single page).</objetivo>
    <tema>Receita de bolos</tema>
    <diretrizes_design>
        <layout>Responsivo e minimalista.</layout>
        <paleta_cores>Marrom e Bege</paleta_cores>
        <tipografia>Sans-serif para títulos, Serif para corpo.</tipografia>
    </diretrizes_design>
    <obrigatoriedades_tecnicas>
        <item>Menu de navegação funcional</item>
        <item>Galeria de imagens</item>
        <item>Rodapé com contato</item>
    </obrigatoriedades_tecnicas>
</tarefa>
