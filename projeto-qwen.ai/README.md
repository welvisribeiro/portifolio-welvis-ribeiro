## 🧠 Aprendizados

- A qualidade da imagem depende diretamente da clareza do prompt  
- Pequenas mudanças no texto geram resultados muito diferentes  
- A IA responde melhor quando recebe contexto detalhado  
- O refinamento iterativo é essencial para alcançar bons resultados

## 🚀 Conclusão

A atividade demonstrou que o uso eficiente de inteligência artificial exige mais do que criatividade: é necessário aplicar estratégia, análise crítica e refinamento contínuo para atingir o resultado desejado.
