# ⚖️ Viés Algorítmico em Classificação de Imagens (Teachable Machine)

## 📝 Descrição do Projeto

Este projeto explora o impacto do viés em modelos de inteligência artificial utilizando a ferramenta Teachable Machine (Google).

Foi desenvolvido um modelo de classificação de imagens com duas categorias: "Perfil Liderança" e "Perfil Operacional", utilizando um conjunto de dados propositalmente enviesado para simular falhas reais em sistemas de IA.

---

## 🎯 Objetivo

- Demonstrar como o viés nos dados afeta decisões algorítmicas  
- Identificar erros de classificação (falsos positivos/negativos)  
- Refletir sobre os impactos sociais da inteligência artificial  

---

## 🧪 Metodologia

### 📌 1. Treinamento do Modelo
- Criação de duas classes:
  - Perfil Liderança
  - Perfil Operacional  
- Uso de imagens com critérios estereotipados

### 📌 2. Teste de Inferência
- Teste com imagens fora do padrão treinado  
- Observação de comportamento do modelo  

### 📌 3. Registro do Erro
- Captura de evidência no momento da falha

## ⚠️ Análise Crítica

📄 **[Ver análise completa](./respostas.pdf)**

### 🧠 Mecanismo do Viés
A seleção restrita de dados limita a capacidade do modelo de generalizar corretamente, fazendo com que ele associe características superficiais a determinadas categorias.

### 💥 Consequência Social
A classificação incorreta pode gerar exclusão, invisibilidade e impacto negativo na percepção de indivíduos ou grupos.

### 🛠 Ação Mitigadora
A aplicação de uma abordagem *human-in-the-loop* permite revisar dados, corrigir falhas e aumentar a equidade do sistema.

---

## 🛠 Ferramenta Utilizada

- Google Teachable Machine

---

## 🚀 Conclusão

O projeto evidencia que modelos de inteligência artificial refletem diretamente a qualidade dos dados utilizados em seu treinamento. Sem diversidade e revisão crítica, esses sistemas podem reforçar vieses e gerar impactos negativos na sociedade.
